# Chain Reaction - Mini App with AI Opponent & Telegram Bot

A beautiful, **single-file** Chain Reaction strategy game with **AI opponent**, hotseat multiplayer, and Telegram integration.

## 🎮 Features
- **Hotseat Multiplayer** (2–4 players)
- **Play vs AI** (Easy / Medium difficulty) — smart chain-reaction AI
- Smooth visual explosions with delays
- Undo last move, player stats, win detection
- Fully responsive (mobile-first)
- Ready as **Telegram Web App** (opens inside chats)

## Quick Start
1. Save the code below as **`index.html`** and open in any browser.
2. Choose mode:
   - **Hotseat** → play with friends by passing device
   - **vs Computer** → play against AI (Medium is challenging!)
3. Enjoy the chains!

**Telegram Bot** (optional but fun):
- Create bot with @BotFather
- Use the `bot.py` code provided earlier (update URL after GitHub Pages deploy)
